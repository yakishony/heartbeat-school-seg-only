מדריך למפתח — Heartbeat PCG Segmentation

מסמך זה מציג את כל קבצי הפרויקט, תוכנם, משתניהם המרכזיים והפונקציות שבהם. הפרויקט מבצע סגמנטציה של הקלטות קול לב (PCG) — סיווג כל נקודת זמן ל-S1, Systole, S2, Diastole או Unannotated — באמצעות רשת עצבית עמוקה מסוג Conv1D-BiGRU (U-Net).

מבנה תיקיות הפרויקט

heartbeat-school-seg_only/ ├── env.py # Configuration — constants and paths ├── download_data_and_annotate_step1.py # Dataset download and per-sample label creation ├── understand_data.py # Exploratory data analysis — plots and statistics ├── run_pipline_analysing_utils.py # Signal processing utilities (normalize, filter, downsample) ├── split_data_into_fixed_length_recordings.py # Split recordings into fixed-length windows ├── run_pipeline_analysing_data_step_2.py # Pipeline orchestration: load → process → save .npy ├── ML_utils.py # Data loading, splitting, class weights, Loss ├── ML.py # Model architecture and training ├── visualize_model.py # Visualization: training curves, confusion matrix ├── plot_utils.py # Plotting utilities (Matplotlib + Plotly) ├── application.py # Gradio web app — user interface ├── checkpoints/ # Saved model checkpoints ├── figures/ # Generated figures └── data_raw.pickle # Pickle cache of raw dataset

אפליקציית ווב (Gradio) לסגמנטציה של הקלטות קול לב. המשתמש מעלה קובץ WAV, האפליקציה מעבדת, מריצה את המודל ומציגה גרף צבעוני אינטראקטיבי.
